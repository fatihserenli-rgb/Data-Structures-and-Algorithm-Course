# Data-Structures-and-Algorithm-Course

# CSE222 - Data Structures and Algorithms

This repository contains my solutions for the assignments of the **CSE222 / BİL505 Data Structures and Algorithms** course. All projects are developed using **Java**, focusing on implementing robust software architectures through advanced Object-Oriented Programming (OOP) techniques and efficient data structures.

---

## Assignment Overviews

### HW1: Embedded System Communication Interface
* **Focus:** Introduction to Java and OOP principles.
* **Description:** Implementation of a specific communication protocol and device hierarchy provided by the instructor. The goal was to manage various devices (sensors, displays, etc.) while adhering to strict architectural constraints and handling errors without terminating the program.

### HW2: Theoretical Algorithm Analysis
* **Focus:** Big-O, Omega, and Theta notations.
* **Description:** A theoretical assignment focused on measuring algorithm analysis skills. It contains no code; instead, it involves mathematical proofs and formal notation to express the asymptotic complexity of various functions.

### HW3: High-Efficiency System Design
* **Focus:** Time complexity optimization.
* **Description:** A reimagining of HW1 with a strict performance requirement. By utilizing specialized data structures and algorithm analysis techniques, all system operations were required to execute in **O(1)** time complexity.

### HW4: Solar System Simulation
* **Focus:** Tree data structures.
* **Description:** A simulation of the solar system using a tree structure. It allows dynamic runtime additions of stars (root nodes), planets, and moons. It features a pathfinding functionality that prints the hierarchical route from the star to any specific celestial body.

### HW5: User-Task Management System
* **Focus:** Priority Queues and Min-Heaps.
* **Description:** Implementation of a task scheduling system where users have specific priority levels. The system ensures that higher-priority tasks are processed first, while tasks with the same priority follow a First-In-First-Out (FIFO) order.

### HW6: High-Performance Dictionary & Autocorrect
* **Focus:** Custom HashMaps, HashSets, and Latency Optimization.
* **Description:** A dictionary application managing over 50,000 words. This task required implementing HashMap and HashSet classes from scratch to ensure search queries and spelling suggestions (additions, deletions, or character swaps) are processed within a **100ms** latency limit.

### HW7 & HW8: Sorting Algorithms & Graph Theory
* **Focus:** Sorting, Matrix Graphs, and Graph Coloring.
* **Description:** A combined project focusing on the implementation of various sorting algorithms and a Matrix-based Graph structure from scratch. These components are integrated with a graph coloring algorithm to demonstrate the practical synergy between sorting and graph traversal.
